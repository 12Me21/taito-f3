Alpha

The fdp uses a somewhat indirect system to choose alpha values.  Each half-pixel has an alpha palette of two alpha values that can be used for it.  You can think of it like this: a color (i.e. as contributed by a layer, and stored in the priority table) consists of 14 bits of data: a 13-bit index into the color palette, and a 1-bit index into the alpha palette, which we call Alpha Select. Alpha values range from 0% to 100%, in steps of 12.5%.
