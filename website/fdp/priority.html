Priority

For each screen pixel, each of the eight graphics layers provides the following data:

  • Color Id (13-bit number: index into palette ram)
  • Alpha Select (1 bit: index into the alpha palette)
  • Enable (1 bit: zero if the pixel is transparent, clipped out, or otherwise disabled)
    it's likely that this is signaled as priority = 0, or blend mode = none
  • Priority Value (4-bit number)
  • Blend Mode (2 bits)
